페이지 구성
 메인페이지
	메인 배너 사진에 스와이퍼기능을 썻음
		(사진 안에 글자를 타이머 기능을 써서 한 글자씩 차례대로 나오게 하는 애니메이션을 만듬)
	메인 배너 밑에 추천 여행지는 aos 기능을 사용해서 스크롤에 따라 등장하는 기능을 사용했다.

 여행지 검색
	카카오맵에서 api를 받아와서 할 예정이였지만 어려워서 일부 지역 검색 기능만 구현

 로그인,회원가입
 	required를 써서 빈칸이 없게 간단하게 이름 이메일 비밀번호만 입력해서 회원가입 창을 만듬

 마이페이지
	회원 정보와 (여행지 검색 기능을 전부 구현하게 되면) 예약 이라던가 찜 목록을 만들어서 연결 하고 사용자가
	여행을 했엇던 곳과 여행지 리뷰 작성 페이지를 만들 예정
	로그인이 되었을때만 표시하고 싶음.
